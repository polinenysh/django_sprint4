# Blogicum - Платформа для ведения блога

## О проекте

Blogicum - это полнофункциональная платформа для ведения блога с системой комментариев, профилями пользователей и категоризацией постов. Проект создан на Django с использованием Class-Based Views и Bootstrap 5.

## Возможности

### Публикации
- **Создание постов** - Авторизованные пользователи могут создавать новые публикации
- **Редактирование и удаление** - Авторы могут редактировать и удалять свои посты
- **Отложенные публикации** - Возможность публиковать посты с будущей датой
- **Снятие с публикации** - Авторы могут скрывать свои посты от других пользователей
- **Категории** - Организация постов по темам и категориям

### Пользователи
- **Регистрация и авторизация** - Стандартная система Django auth
- **Профили пользователей** - Просмотр профилей других пользователей
- **Редактирование профиля** - Изменение имени, email и других данных
- **Личный кабинет** - Авторы видят все свои посты, включая неопубликованные

### Комментарии
- **Добавление комментариев** - Авторизованные пользователи могут комментировать посты
- **Редактирование и удаление** - Авторы могут редактировать и удалять свои комментарии

### Администрирование
- **Панель администратора** - Управление пользователями, постами, категориями, комментариями
- **Модерация контента** - Снятие постов с публикации, управление комментариями
- **Создание категорий** - Только администраторы могут создавать новые категории
